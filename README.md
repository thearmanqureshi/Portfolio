# Personal Portfolio Website
This repository contains the code for my personal portfolio website, built with HTML, CSS, and vanilla JavaScript. It is designed to showcase my skills, projects, and professional experience as a web developer through a clean, responsive, and user-friendly interface.

The live website can be viewed at: **[thearmanqureshi.me](https://www.thearmanqureshi.me)**

## Key Features

-   **Responsive Design:** Adapts seamlessly to all screen sizes, ensuring a great user experience on desktops, tablets, and mobile devices.
-   **Smooth Scrolling:** Implemented using Lenis.js to provide an enhanced and fluid browsing experience.
-   **SEO Optimized:** Configured with appropriate meta tags, a `sitemap.xml`, and `robots.txt` for better search engine visibility and indexing.
-   **Experience & Education:** Presents my professional and academic background in a clean, timeline-style format.
-   **Project Showcase:** Displays a curated selection of my featured projects, complete with descriptions, live demo links, and links to their GitHub repositories.
-   **AJAX Contact Form:** Includes a functional contact form that uses the Web3Forms API to send messages asynchronously without page reloads.

## Installation

To run the project locally, follow these steps:

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/thearmanqureshi/Portfolio.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd Portfolio
    ```
3.  Open the `index.html` file in your preferred web browser.

## Deployment

This project can be easily deployed using Vercel.

1.  Install the Vercel CLI globally:
    ```sh
    npm install -g vercel
    ```
2.  Log in to your Vercel account:
    ```sh
    vercel login
    ```
3.  Deploy the project from the root directory:
    ```sh
    vercel
    ```
